# SX4 Fahrplan

Mehr allgemeine Info: https://opensx.net/sx4

## Installation

Nach Download von *SX4.jar* von github, Verzeichnis 

    https://github.com/michael71/SX4/releases

starten mit 
    
    java -jar SX4.jar  [Startoptionen...]

(Java-8 oder Java-11 muss installiert sein)
